Raspberry Pi DHT11 Sensor Project

This project uses a Raspberry Pi and a DHT11 sensor to collect temperature and humidity data, which is displayed on a web page using Flask.

The DHT11 sensor is a low-cost digital temperature and humidity sensor that can be connected to a Raspberry Pi. The Raspberry Pi runs a Flask web server that reads the sensor data and serves it to the web page.

Requirements

  • Raspberry Pi
  • DHT11 sensor
  • Python 3
  • Flask

Instructions

  1. Connect the DHT11 sensor to the Raspberry Pi.
  2. Install Python 3 and Flask on the Raspberry Pi.
  3. Download the project code from GitHub.
  4. Run the Flask web server.
  5. Open a web browser and navigate to the Raspberry Pi's IP address to view the sensor data.

Code

Here's an example of the Python code used to read the sensor data:

import Adafruit_DHT

# Set sensor type and GPIO pin
sensor = Adafruit_DHT.DHT11
pin = 4

# Read sensor data
humidity, temperature = Adafruit_DHT.read_retry(sensor, pin)

# Print sensor data
print('Temperature: {}C'.format(temperature))
print('Humidity: {}%'.format(humidity))
